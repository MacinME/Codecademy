HTML Bests Practicing
N. Description Example Critical
1 Using semantic HTML can be very useful for user experience and developers, it makes it easier to read and for SEO (Search Engine Optimization) Use as posible as you can <section> tag, instead of <div>, or other types of tags available.
2 Add the specific code into your HTML document to instructs the browser to use the last version of HTML At the top of the HTML document aggregate the next line of code: <!DOCTYPE html>
3 Indentation to HTML code it will be easier and understandable for other developers who will interact with your code You can use some very useful extension on vscode like: Prettier
CSS Bests Practicing
N. Description Example Critical
1 Use an External CSS stylesheet to style your project. You can create a style.css file within the directory you are working on. Additionally, link the CSS file from HTML document.
2 It's hardly recommended to reset agent-values or the default styles that browsers apply. One simple CSS rule could be reseting the margin and padding for complex layouts: *{ margin: 0; padding: 0, box-sizing: border-box; }
3 Use class or element selector as possible as you can, it will help your code to override future declarations that might you set, this is very helpful because we can easily make changes, avoiding the !important keyword value, one tip can be useful when we are using CSS frameworks, but don't use it frequently' .class-h1 {}, h1 { }, instead of #h1 {}
